Overview: Analysis on Data Tracked by Google Analytics

Analysis of historical data from the COPA database retrieved in Sep. 2019.

Background Information: When we explored the COPA database which retrieved during Sep. 2019, we found there is no relevant record about COPA members' log data (information about COPA members' each visit and how they behaved on the website). To study more about COPA members visiting behaviors and also to generate more actable suggestion for future improvement, we received the new website's Google Analytics access. We explored Google Analytics' data 5 dimensions:

  • The website traffic
  • The search queries COPA members used in the website
  • Users characteristics
  • Acquisition: the source of the traffic
  • Behavior

    • Analysis Purpose: This analysis and dashboard aim to provide COPA members, website administrators, and management team a comprehensive understanding of the website performance since the new website migration (April 2nd, 2020). The analysis summarized the data at a high-level and hope to bring some useful findings that can help with future improvement.

    Findings :

  • According to the data we explored, COPA members are adapting using the new website.
  • The search pages' metrics show that most search actions did not lead to an exit, which is a good thing.
  • COPA is a global organization that attracts users from 97 countries/regions. Most users come from the United States. However, users from some non-English speaking countries tend to have a higher bounce rate.
  • For the devices that COPA members used to visit the website, mobile users have a higher bounce rate and lower session duration. More specifically, old models of mobile phones have high bounce rate.
  • Almost 70% of COPA members visited the website by URLs directly. Compared to the industry benchmark, COPA has much fewer visitors who reached the website through social media reference.
  • While adapting to the new website, many users reached the 404 error page and some other password resetting pages. In these pages, people tend to have a higher probability to exit the website.

    • Opportunities:

  • Google Analytics provides the website internal search tracker, which can track how often people use the search engine, how long people spend on searching and search refinements, and how depth the search is. To access this service, COPA can set up the tracker in the Admin view setting page by turning on the search tracker button.
  • To study a specific function of the website, such as the search engine performance, COPA can set up Goal or Event on Google Analytics. By doing so, we can study the conversion rate of these events.
  • Google Analytics automatically groups web pages, but the grouping does not clean. COPA can further regroup pages into different forum categories to study the users' behavior flow (clickstream).
  • Google Analytics would automatically generate some insights. Among those insights, there are some week over week reports that are useful to compare the website performance week over week.
  • Google Analytics provides a benchmark comparison for 1,600 industries. By choosing the right industry, COPA can compare the website to other websites in the same industry.

    • Limitation: Google Analytics is tracking the metrics of the website. More efforts of utilizing these metrics are needed. Also, Google Analytics identifies users by their browser cookies, so it cannot recognize COPA members. Ad hoc analysis based on the COPA database is the complement of Google Analytics because it can generate other customized analyses and have information about the content of posts.
