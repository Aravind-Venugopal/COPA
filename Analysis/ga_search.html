Internal Search Queries

Search queries used by the users who visited the COPA website from April 2nd to 18th.

Google Analytics provides a site search tracker for website owners to study what users searched on the website, the time spent on the search page, and the search depth. We recommend the COPA website administrator try to use this service to learn more about the search function on the website by turn on the button on the view setting of Google Analytics. Currently, we search-relevant URL paths in among pages that users visited. Here are the search queries used by COPA members to find the post they want. We extracted data from Google Analytics and plot the interactive dashboard below. By clicking the point or the sorting icon of the bar chart, you can see the tooltip and sort the search queries by unique pageviews or the %Exit (the probability of exit the website on the search query page). The search query with high %Exit may be the query that COPA members cannot get the relevant information easily.
