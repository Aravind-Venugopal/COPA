User Behavior Analysis

This section includes the basic analysis of user behavior on the COPA website.

Since the launch of the new COPA site, it has 170k page views with nearly 60% of them are unique page views. Through the line chart, it doesn't change too much day by day with only a small spike over the weekend.

The below graph shows the content drill-down information. The website forum section is the core function of COPA, which shows 40% percent of the user traffic toward the Forum content page. The main site also shows a large number of user traffic, these users are coming in as an organic search where they are browsing through the COPA main website. The deliverable of this section is showing the main traffic of the user and in the near future, spreading out any needed information or advertisement could in result show the information or advertisement on the main site and the forum post site only.

To have an intuitive sense, we extracted the data to build up a like-BCG Matrix for these pages. The median line divided the chart into four modules, and each point represents a page with the page title. By hovering on the point, you can see the page title, %Exit (the probability that users exit the website on this page), and the unique pageviews. The blue shadow the pages within the quantile of the median. This means that the page out of this area should be focused heavily. In the danger zone, these pages have many pageviews but also have higher %Exit. There is a page for 404 error, which may be caused by users who used the old website link to reach the new website page. We can see the website migration would change the URL of the webpages, adding inconvenience to users who saved the old website's links.
