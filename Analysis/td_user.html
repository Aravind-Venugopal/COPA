COPA Members

Number of COPA members visited the website - data from the COPA database retrieved in Sep. 2019.

In the database, there is information about COPA members' names, emails, and credit card information. We did not find members' demographic data in the database, but in Google Analytics, there is one section can track users' gender and age. If need, the COPA website administrators can try to set it up.

There are about 22K+ COPA members in history, the members who were still active during 2019 are only 3,090 people. The number of COPA members is increasing slowly. Every month, there are about 103 new members join in COPA. On each day, new COPA members' visits only contribute very small proportion (often less than 1%) of the traffic.
