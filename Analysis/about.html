COPA Initialtive Practicum Project

Leveraging data power to boost website user experience and finding insights for the future improvement.

The COPA project is a cooperative project between Cirrus Owners & Pilots Association (COPA) and MSBA students from the University of California, Davis. The MSBA student team will utilize the skills and knowledge related to business analytics to help COPA solve the problems. The objectives of this project are helping COPA members find needed information and providing better user experience. To achieve the objectives, the COPA project will involve three deliverables:

  1. Internal search engine optimization
    2. Improve the internal search engine so that members can search for the most relevant information or forum posts easily. With the internal search engine optimization, COPA members can search for the needed or valued information efficiently and accurately.
  2. Data silo integration
    3. Integrate high-quality and useful external information related to GA and safety topics into the website. The data silo integration will provide high valued information both from the internal stack information database and external authorized database.
  3. Data analytics dashboard
    4. Provide insights through data analysis to website administrators for future improvement. COPA was founded in 2001 and has 1.1M+ posts in the forum. We can dig in this huge text data and find out some insights related to potential future improvement for the website administrators.

Search Result Re-ranking Model

To improve the search results on the COPA website, we built up a re-ranking model, which re-ordered the search results by the recency, the authority of the author, and the relevancy between the content and the search query.
